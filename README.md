# Poker Solver (Equity Calculator)

Minimal, fast, and simple equity calculator. Enter a hero hand and a villain range, run Monte Carlo simulations, and get an equity estimate. Ships with:
- **Web app** (FastAPI + Uvicorn) – simple UI that feels like a real app.
- **CLI tool** – quick terminal runs for scripted checks.

> Note: this is an **equity calculator**, not a full GTO solver like GTO Wizard.
